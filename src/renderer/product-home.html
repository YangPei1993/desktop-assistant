manman

把桌面协作这件事，做得又稳又快。

manman 是一个常驻在 macOS 桌面的智能助手，统一了聊天、自动化执行和实时观察三种工作方式。

查看能力 使用场景
  • 多通道运行：CLI / API 一键切换
  • 自动化任务：规划 + 执行直至完成
  • Live Watch：持续观察并主动提醒

品牌识别：留白、克制、专注。

统一输入

支持文本、语音、图片与文件，所有输入在同一对话上下文中自然衔接。

自动化执行

针对桌面任务自动分步处理，减少重复点击与来回切换窗口的成本。

实时协同

Live Watch 在你工作时持续观察变化，主动指出风险、问题与下一步建议。

适合这些高频场景

当你需要边看边做、边做边确认时，manman 能把“思考”和“执行”连成一条顺滑路径。
